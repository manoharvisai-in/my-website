Introduction & HistoryWho invented it · What it is
Historical Milestone
  • Transistor was invented on December 23, 1947 at Bell Laboratories (research arm of AT&T).
  • Inventors: William Shockley, John Bardeen, and Walter Brattain (Nobel Prize 1956).
  • Name origin: TRANSfer + reSISTOR = TRANSISTOR (transfers signal from low-R to high-R path).
What is a BJT?

A Bipolar Junction Transistor (BJT) is a three-terminal semiconductor device made of two back-to-back PN junctions. Current depends on both majority and minority carriers — hence "bipolar."

Key Fact
BJT is a current-controlled device — output current is controlled by input current.
Two Types of BJT
TypeStructureMajority Carriers in Base
NPNN–P–NHoles
PNPP–N–PElectrons
Arrow Rule
Arrow is always on the Emitter. It always points toward the N-type region.
PNP: E→B  |  NPN: B→E (conventional current direction)
Applications
  • Amplifier (analog electronics — active region)
  • Switch (digital electronics — saturation & cutoff regions)
  • Oscillator circuits, computers, satellites, communication systems

"Heavy–Light–Moderate" → Emitter–Base–Collector (E-B-C doping order).
Mnemonic: "Extremely Low Middle"
Important Ratios & Facts
  • Ratio of total device width to base width = 150:1
  • Emitter-to-outer doping ratio ≈ 10:1 (keeps base lightly doped)
  • BJT has two PN junctions: Emitter-Base (EB) and Collector-Base (CB)
  • Two-diode analogy: back-to-back diodes (but transistors cannot be replaced by two diodes!)

Current Components & Key EquationsKCL · Leakage · α, β, γ
Basic KCL Relation
IE = IC + IB KCL at BJT
IE = Emitter current   IC = Collector current   IB = Base current

Note: IE is the largest, IB is the smallest (≈1–5% of IE).

CB Current Equation (with leakage)
IC = α·IE + ICBO
IC = [α/(1−α)]·IB + [1/(1−α)]·ICBO
ICBO = reverse saturation (leakage) current in CB
CE Current Equation (with leakage)
IC = β·IB + ICEO
ICEO = ICBO / (1−α) = (1+β)·ICBO
ICEO = leakage current in CE  (≫ ICBO, so CE is more temperature-sensitive)
CC Current Equation
IE = (1+β)·IB + ICCO
ICCO = leakage current in CC = (1+β)·ICBO

All Formulas at a GlanceRelations between α, β, γ
α ↔ β ↔ γ Relations ★ Most Tested
β = α / (1 − α)
α = β / (1 + β)
γ = 1 / (1 − α) = 1 + β
α = (γ − 1) / γ Derivable from IE=IC+IB
Trick: If α is given → β = α/(1−α)  |  If β is given → α = β/(β+1)
γ is always = 1 + β — just add 1 to β!
Leakage Current Relations
ICEO = ICBO / (1 − α)
ICEO = (1 + β) · ICBO
ICCO = ICEO = (1 + β) · ICBO
⚠ Warning
ICEO is (1+β) times larger than ICBO. This makes CE configuration more sensitive to temperature than CB.
Fixed Bias Equations
IB = (VCC − VBE) / RB
IC = β · IB
VCE = VCC − IC · RC
S (stability) = 1 + β Fixed / Base Bias
VBE = 0.7V (Silicon), 0.3V (Germanium)
DC Load Line Equations
VCE = VCC − IC · RL
Point A: IC = 0 → VCE = VCC
Point B: VCE = 0 → IC = VCC / RL

CB CharacteristicsInput & Output — Operating Regions
CB Input Characteristics (IE vs VBE)
  • Plot of input current (IE) vs input voltage (VBE) for constant VCB.
  • Shape is identical to diode V-I curve (forward biased EB junction).
  • For constant VBE: IE increases with increase in VCB (due to base width reduction).
  • For VBE > cut-in voltage (≈0.7V): Input resistance is very small.
  • Curves shift right as VCB increases (Early effect).
CB Output Characteristics (IC vs VCB) FAQ
RegionEB JunctionCB JunctionKey Property
ActiveForwardReverseIC ≈ IE (amplifier region)
SaturationForwardForwardLeft of VCB=0, above IE=0
CutoffReverseReverseBelow IE=0; IC = ICBO
Breakdown—High reverse VAvoid! Transistor may be destroyed
Active Region Key Points
IC controlled by IE, almost independent of VCB. IB is very small, so IC ≈ IE.
Base Width Modulation (Early Effect) Important
  • Increasing reverse bias on CB junction → depletion region widens into base → effective base width decreases.
  • This is called Early Effect or Base Width Modulation.
  • Reduced base width → fewer recombinations → IE increases for same VBE.
  • If base width → 0: punch-through occurs. Large IC flows; transistor may be destroyed.

Load Line, Q-Point & Fixed BiasingDC Analysis · Stability · Thermal Runaway
DC Load Line

A DC load line is a line that represents all possible DC operating conditions (IC, VCE) of the circuit, drawn on the output characteristics graph.

How to Draw DC Load Line
1. Set IC = 0 → Point on X-axis: VCE = VCC
2. Set VCE = 0 → Point on Y-axis: IC = VCC/RL
3. Join the two points → DC Load Line
Q-Point (Quiescent Point)
  • The intersection of the DC load line and the transistor output characteristic curve for a given IB.
  • Represents the DC operating point (no AC signal applied).
  • For faithful amplification: Q-point must be in the middle of the active region of the load line.
  • If Q shifts to top → positive peak clipped. If Q shifts to bottom → negative peak clipped.
Thermal Runaway

As temperature increases → ICBO doubles every 10°C → IC increases → power dissipation increases → temperature increases further. This cumulative process can destroy the transistor.

Three Factors that Shift Q-Point
1. ICO doubles for every 10°C rise in temperature.
2. VBE decreases by 2.5 mV/°C.
3. β (hFE) increases with temperature.
Stability Factors S, S', S''
S = ∂IC/∂ICO (keeping VBE, β constant)
S' = ∂IC/∂VBE (keeping ICO, β constant)
S'' = ∂IC/∂β (keeping ICO, VBE constant)

Lower S = more stable circuit. Ideal value: S = 1.

Fixed Bias (Base Bias) Circuit Important
  • Single VCC for both collector and base via RB.
  • VCC = IB·RB + VBE → IB = (VCC − VBE)/RB
  • Stability factor: S = 1 + β (very poor — large, so unstable).
✓ Merits
Simple circuit. Few components. Easy to adjust Q-point by changing RB.
✗ Demerits
Very poor stability (S = 1+β). Q-point shifts with temperature and transistor replacement.

Solved NumericalsStep-by-step with tricks
Solved Problem #1
Calculate IC and IE for a transistor with α = 0.98 and IB = 100 μA. Also find β.
① Find β from α: β = α/(1−α) = 0.98/(1−0.98) = 0.98/0.02 = 49
② IC = β × IB = 49 × 100 μA = 4900 μA = 4.9 mA
③ IE = IC + IB = 4.9 mA + 0.1 mA = 5 mA
✓ Answer: IC = 4.9 mA, IE = 5 mA, β = 49
Solved Problem #2
NPN transistor α = 0.98, IE = 3 mA, ICO = 10 μA (CB mode). Find IB and IC.
① IC = α·IE + ICBO = 0.98 × 3 mA + 10 μA = 2.94 mA + 0.01 mA = 2.95 mA
② IB = IE − IC = 3 mA − 2.95 mA = 0.05 mA = 50 μA
✓ Answer: IC = 2.95 mA, IB = 50 μA
Solved Problem #3
Transistor α = 0.975, Ico = 10 μA in CE mode. Find β. If IB = 250 μA, find IE and IC.
① β = α/(1−α) = 0.975/(1−0.975) = 0.975/0.025 = 39
② ICEO = (1+β)·ICBO = 40 × 10 μA = 400 μA = 0.4 mA
③ IC = β·IB + ICEO = 39 × 0.25 mA + 0.4 mA = 9.75 + 0.4 = 10.15 mA
④ IE = IC + IB = 10.15 + 0.25 = 10.4 mA
✓ Answer: β = 39, IC = 10.15 mA, IE = 10.4 mA
Solved Problem #4 — Fixed Bias
Fixed bias: RB = 240 kΩ, RC = 2.2 kΩ, β = 99, VCC = 12 V, VBE = 0.7 V. Find IB, IC, VCE.
① IB = (VCC − VBE) / RB = (12 − 0.7) / 240k = 11.3 / 240000 = 47.08 μA
② IC = β × IB = 99 × 47.08 μA = 4.66 mA
③ VCE = VCC − IC·RC = 12 − 4.66m × 2.2k = 12 − 10.25 = 1.75 V
✓ Answer: IB ≈ 47 μA, IC ≈ 4.66 mA, VCE ≈ 1.75 V
Solved Problem #5 — CEO Leakage
NPN transistor α = 0.98, Ico (CB) = 10 nA. Find leakage current in CE mode (ICEO).
① β = α/(1−α) = 0.98/0.02 = 49
② ICEO = (1+β) × ICBO = (1+49) × 10 nA = 50 × 10 nA = 500 nA = 0.5 μA
✓ ICEO = 500 nA (50× amplified compared to CB leakage!)

Common Mistakes, FAQs & Exam TipsScore higher by avoiding these
❌ Common Mistakes
✗Confusing α with β. Remember: α = β/(1+β) < 1; β = α/(1−α) > 1.
✗Forgetting to add ICEO when leakage is given in CE mode: IC = β·IB + ICEO, NOT just β·IB.
✗Using wrong VBE: Use 0.7V for Silicon, 0.3V for Germanium.
✗Thinking two diodes back-to-back = transistor. They are NOT equivalent.
✗Saying CE has 0° phase shift — it is 180° (it INVERTS the signal).
✗Drawing DC load line incorrectly: X-intercept = VCC; Y-intercept = VCC/RL.
✗Confusing ICBO (CB leakage) with ICEO (CE leakage). ICEO = (1+β)·ICBO.
❓ Frequently Asked Questions
  • Why is CE most commonly used? — Both current AND voltage gain are high, giving maximum power gain.
  • Why is base kept thin and lightly doped? — To reduce recombination; almost all carriers reach collector.
  • What is punch-through? — When base width → 0 due to excessive reverse bias; large IC flows, transistor destroyed.
  • Why is Fixed Bias poor? — Stability factor S = 1+β is very high; Q-point shifts with temperature.
  • What happens at Q-point if biased at top of load line? — Positive half-cycle of output gets clipped.
  • Why does ICEO ≫ ICBO? — The (1+β) amplification factor, so CE config is far more temperature-sensitive.
✍ Exam Presentation Tips Score Tips
  • Always state given values clearly at start: "Given: α = 0.98, IB = 100 μA"
  • Write formula first, then substitute: "IC = β × IB = 49 × 100 μA"
  • Box your final answers for easy examiner identification.
  • For characteristics questions: Draw the graph with labeled axes, indicate regions (Active/Saturation/Cutoff).
  • For biasing questions: Draw the circuit, write KVL equation, then solve.
  • Always check units: mA, μA, kΩ — unit errors = mark deductions.

⚡ Mega Revision Sheet

Everything you need to revise the night before your exam

🔬 BJT Basics

  • Invented: Dec 23, 1947 at Bell Labs
  • Inventors: Shockley, Bardeen, Brattain
  • Current-controlled device
  • E: heavily doped | B: lightly | C: moderate
  • Base/total width ratio = 1/150
  • Two PN junctions; "bipolar" = both carriers

⚡ Key Formulas

  • I_E = I_C + I_B
  • α = I_C/I_E (0.95–0.995)
  • β = I_C/I_B (>1)
  • γ = I_E/I_B = 1+β
  • β = α/(1-α)
  • α = β/(1+β)
  • I_CEO = (1+β)·I_CBO

📊 CB vs CE vs CC

  • CB: Very low R_in, Very high R_out, α<1
  • CE: Low R_in, High R_out, high β, 180° shift
  • CC: High R_in, Low R_out, γ=1+β, 0° shift
  • CE has maximum power gain → most used
  • CC = emitter follower (impedance matching)
  • CB: HF applications (no Miller effect)

📉 Operating Regions

  • Active: EB fwd + CB rev → Amplifier
  • Saturation: Both fwd → Closed switch
  • Cutoff: Both rev → Open switch
  • Breakdown: Excess V_CB → Destroy!
  • Punch-through: Base width → 0
  • Early effect = Base width modulation

📏 Load Line & Q-Point

  • Load line: V_CE = V_CC − I_C·R_L
  • X-intercept: V_CC (when I_C=0)
  • Y-intercept: V_CC/R_L (when V_CE=0)
  • Q-point = intersection with char. curve
  • Best Q: middle of load line
  • Q shifts → signal clipping/distortion

🔋 Fixed Bias

  • I_B = (V_CC - V_BE)/R_B
  • I_C = β·I_B
  • V_CE = V_CC - I_C·R_C
  • Stability S = 1+β → POOR
  • V_BE = 0.7V (Si), 0.3V (Ge)
  • Merit: simple | Demerit: unstable

🌡 Thermal Stability

  • I_CBO doubles per 10°C rise
  • V_BE decreases 2.5 mV/°C
  • β increases with temperature
  • Thermal runaway → transistor damage
  • S, S', S'' = stability factors
  • Lower S = more stable bias circuit

🧠 Quick Memory Tricks

  • β = α/(1-α) → "alpha divided by its complement"
  • γ = 1+β → "just add 1 to beta"
  • CE: 180° → "CE Certainly inverts"
  • Active: fwd-rev → "normal working"
  • I_CEO = (1+β)·I_CBO → amplified leakage
  • I_E is always LARGEST of three currents
