1 Operational Amplifier (Op-Amp)
Introduction · Characteristics · CMRR · Virtual Ground · Configurations

What is an Op-Amp? Definition

  • General-purpose IC developed by Robert Widlar in 1964.
  • Can perform mathematical operations: summation, subtraction, integration, differentiation.
  • "Op" = operational → refers to performing operations.
  • Most popular IC: LM741 (8-pin DIP).
  • Low cost, easy to use, temperature stable.
  • Only terminal properties needed — internal circuit not required.

Applications Real World

  • Signal amplification
  • Oscillators
  • Filters (active filters)
  • Voltage regulators
  • ADC / DAC converters
  • Calculators & instrumentation
  • Summing / Difference amplifiers

CMRR — Common Mode Rejection Ratio Formula

Op-Amp amplifies the difference between V₁ and V₂. CMRR measures how well it rejects the common (same) signal.

CMRR
CMRR = Aᵈ / Aᶜ
CMRR (dB) = 20 log(Aᵈ/Aᶜ)
Vd = V₁ − V₂  (differential signal)
Vc = (V₁ + V₂)/2  (common signal)
Aᵈ = differential gain  |  Aᶜ = common mode gain
Ideally CMRR = ∞ (Aᶜ = 0)
💡
Exam Tip: Higher CMRR = better op-amp = more noise rejection. Always express in dB in exams.

Virtual Ground Concept

  • In an inverting amplifier with feedback, the (−) input is NOT physically connected to ground.
  • But due to infinite gain and feedback, the voltage at (−) ≈ 0 V → acts as ground.
  • This point is called Virtual Ground.
  • No current flows into the op-amp input (Zin = ∞).
⚠️
Common Mistake: Students think virtual ground means the inverting pin is actually shorted to ground. It is NOT — it's just at 0 V due to feedback.

Op-Amp Configurations — Formulas All Types

① Inverting Amplifier

Gain
Av = −Rf / R1
Negative sign → 180° phase shift
Rf = feedback resistor, R1 = input resistor

② Non-Inverting Amplifier

Gain
Av = 1 + Rf / R1
Always ≥ 1 | No phase inversion

③ Unity Gain Buffer

Gain
Av = 1
Output connected directly to (−) input
Vout = Vin | Used for impedance isolation

④ Summing Amplifier

Output
Vout = −Rf(V1/R1 + V2/R2)
If R1=R2=Rf: Vout = −(V1+V2)

⑤ Differential Amplifier

Output
Vout = (R2/R1)(V2 − V1)
If R1=R2: Vout = V2 − V1  |  Amplifies difference, rejects common mode

⑥ Comparator

Logic Output
Vout = +Vsat  if  V+ > V−
Vout = −Vsat  if  V+ < V−
No feedback | Operates in open-loop | Digital output
🌟
Most Important Points: Inverting amp = negative gain. Non-inverting amp = gain ≥ 1. Buffer = gain exactly 1. Summing = weighted adder. Comparator = no feedback, outputs HIGH or LOW only.

Half Adder & Half Subtractor Combinational

Half Adder

Adds two 1-bit numbers → Sum (S) + Carry (C)

Equations
S = A⊕B = A'B + AB'
C = A·B
Gates needed: 1 XOR + 1 AND

Half Subtractor

Subtracts two 1-bit numbers → Difference (D) + Borrow (B₀)

Equations
D = A⊕B = A'B + AB'
B₀ = A'·B
Gates needed: 1 XOR + 1 AND + 1 NOT
⚠️
Key Difference: Half Adder and Half Subtractor have the same Sum/Difference equation (XOR), but Carry = AB while Borrow = A'B.

Multiplexer (MUX) Data Selector

  • Routes 2ᴺ inputs → 1 output using N select lines.
  • Acts as a digital switch.
  • 2×1 MUX equation: Z = S'·I₀ + S·I₁
  • When S=0 → Z=I₀  |  When S=1 → Z=I₁
  • MUX can implement any Boolean function → universal circuit.

Flip-Flop Basics Memory Element

  • A flip-flop stores 1 bit of data — either 0 or 1.
  • It is a bistable device — 2 stable states.
  • Types: SR, JK, D, T, MS flip-flops.
  • Latch: Level triggered (responds to input level), NO clock needed.
  • Flip-Flop: Edge triggered (responds to clock edge — rising or falling).
📌
Exam Key: Latch = level triggered, no clock. Flip-Flop = edge triggered, needs clock.

SR Latch NAND Based

Built with 2 cross-coupled NAND/NOR gates. Has S (Set) and R (Reset) inputs.

⚠️

JK Flip-Flop Improved SR

JK FF is an SR FF with added feedback — eliminates the invalid state. J=Set, K=Reset.

FET Types — Classification Tree

JFET — Junction Field Effect Transistor Theory

Terminals

  • Gate (G): Connected to p-type material, controls current.
  • Drain (D): Positive terminal — where electrons are deficit.
  • Source (S): Negative terminal — electron accumulation.

Operation (n-channel)

  • VGS=0, VDS applied → current ID flows.
  • As VDS↑ → depletion region widens → resistance↑.
  • At pinch-off (VP): ID saturates = IDSS.
  • VGS< 0 → channel narrows → ID decreases.
  • VGS = VGS(off) → ID = 0 (cutoff).

3 Operating Regions

①
Ohmic/Linear Region: VDS < VP, ID depends on VDS — acts as voltage-controlled resistor.
②
Saturation/Pinch-off Region: VDS ≥ VP, ID ≈ constant = IDSS. Active amplification region.
③
Breakdown Region: VDS > VDSmax → ID increases uncontrollably → JFET destroyed!

JFET Key Formulas Must Know

Shockley's Equation
ID = IDSS(1 − VGS/VP)²
ID = Drain current at given VGS
IDSS = Max drain current (VGS=0)
VP = Pinch-off voltage = VGS(off)
When VGS=0: ID=IDSS
When VGS=VP: ID=0
Dynamic Resistance
rd = ΔVDS / ΔID |VGS=const
Reciprocal of slope in drain characteristics
Transconductance
gm = ΔID / ΔVGS |VDS=const
Rate of change of ID w.r.t. VGS
Amplification Factor
μ = ΔVDS / ΔVGS |ID=const = gm × rd
μ = amplification factor  |  gm = transconductance  |  rd = dynamic resistance

MOSFET — Metal Oxide Semiconductor FET D-type vs E-type

Also called IGFET (Insulated Gate FET). Gate is insulated from channel by SiO₂ layer → infinite input impedance of gate.

VGS=0 → ID=IDSS (depletion mode)
VGS>0 → ID>IDSS (enhancement mode)

N Solved Numericals
Step-by-step solutions with exam tricks
Q1. A JFET amplifier has VGS changed from −1V to −1.5V (VDS constant) causing ID to drop from 7mA to 5mA. Find gm and amplification factor if rd = 200kΩ.
Step 1: Transconductance: gm = ΔID / ΔVGS
ΔID = 7mA − 5mA = 2mA
ΔVGS = −1.5V − (−1V) = −0.5V
gm = 2mA / 0.5V = 4 mA/V
Step 2: Amplification factor: μ = gm × rd = 4×10⁻³ × 200×10³
∴ gm = 4 mA/V  |  μ = 800
Q2. A JFET has ID = 6mA, VP = −8V, VGS = −4.7V. Find IDSS.
Using Shockley's equation: ID = IDSS(1 − VGS/VP)²
6mA = IDSS × (1 − (−4.7)/(−8))²
6mA = IDSS × (1 − 0.5875)²
6mA = IDSS × (0.4125)² = IDSS × 0.1702
IDSS = 6mA / 0.1702
∴ IDSS ≈ 35.3 mA
Q3. JFET has IDSS = 40mA, VP = −6V, VGS = −4.5V. Find ID.
Using: ID = IDSS(1 − VGS/VP)²
ID = 40mA × (1 − (−4.5)/(−6))²
ID = 40mA × (1 − 0.75)² = 40mA × (0.25)²
ID = 40mA × 0.0625
∴ ID = 2.5 mA
Q4. Convert (315)₁₀ to Hexadecimal.
315 ÷ 16 = 19 remainder 11 (B)
19 ÷ 16 = 1 remainder 3
1 ÷ 16 = 0 remainder 1
Read remainders from bottom to top (MSD → LSD)
∴ (315)₁₀ = (13B)₁₆

⚠️ Common Mistakes Avoid These

  • Confusing virtual ground with actual ground.
  • Forgetting the negative sign in inverting amplifier gain.
  • Using S=R=1 in SR flip-flop calculations.
  • Confusing latch (level triggered) with flip-flop (edge triggered).
  • Wrong sign of VGS in Shockley's equation (VP is negative for n-channel).
  • Forgetting to read remainders from bottom to top in conversions.
  • Confusing XOR (different inputs→1) with XNOR (same inputs→1).

⚡ Op-Amp — Quick Recall

  • Developed by Robert Widlar (1964) | IC: LM741
  • Ideal: Zin=∞, Zout=0, Av=∞, BW=∞, CMRR=∞
  • Two golden rules: V⁺=V⁻ (virtual short), I_in=0
  • Inverting Amp: Av = −Rf/R1 (180° phase shift)
  • Non-Inverting: Av = 1 + Rf/R1 (always ≥ 1)
  • Unity Buffer: Av = 1 (impedance isolation)
  • Summing: Vout = −Rf(V1/R1 + V2/R2)
  • CMRR = Ad/Ac = 20log(Ad/Ac) dB → higher is better
  • Comparator: No feedback, Vout = ±Vsat
  • Virtual ground: (−) input at 0V due to feedback

💻 Digital Systems — Quick Recall

  • Binary=base 2 | Octal=base 8 | Hex=base 16
  • Decimal→Binary: repeated division by 2, read up
  • Binary→Octal: group 3 bits | →Hex: group 4 bits
  • Boolean: A+A'=1 | A·A'=0 | A+1=1 | A·0=0
  • De Morgan: (AB)' = A'+B' | (A+B)' = A'·B'
  • AND: output 1 only when all inputs 1
  • OR: output 0 only when all inputs 0
  • XOR: output 1 when inputs DIFFER
  • NAND/NOR: Universal gates (can make all others)
  • Half Adder: S=A⊕B, C=AB | Subtractor: D=A⊕B, B₀=A'B
  • 2×1 MUX: Z = S'I₀ + SI₁

🔁 Sequential Circuits — Quick Recall

  • Sequential = present input + past states (has memory)
  • Latch: level triggered, no clock needed
  • Flip-flop: edge triggered, needs clock
  • SR Latch: S=set Q=1 | R=reset Q=0 | S=R=1: INVALID!
  • SR FF: same as latch but gated by clock
  • JK FF: J=K=1 → TOGGLE (fixes SR invalid state)
  • JK = SR + feedback loops to eliminate undefined state
  • Bistable: 2 stable states (0 or 1)
  • Applications: Counters, registers, memory cells
  • SR forbidden combo: S=R=1 → avoid always!

📡 FET & MOSFET — Quick Recall

  • FET = voltage controlled | BJT = current controlled
  • FET: unipolar | BJT: bipolar
  • FET: higher Zin, less noise, no thermal runaway
  • JFET terminals: Gate, Drain, Source
  • n-Ch JFET: VGS=0 → max ID=IDSS; VGS↓ → ID↓
  • Pinch-off (VP): ID saturates = IDSS
  • VGS=VGS(off)=VP → ID=0 (cutoff)
  • Shockley: ID = IDSS(1 − VGS/VP)²
  • gm = ΔID/ΔVGS | rd = ΔVDS/ΔID | μ = gm×rd
  • D-MOSFET: channel exists at VGS=0; can deplete or enhance
  • E-MOSFET: NO channel at VGS=0; needs VGS>VT
  • E-MOSFET: ID = k(VGS−VT)² | VDSsat = VGS−VT
  • MOSFET also called IGFET; SiO₂ insulates gate
  • MOSFET advantages: small size, low power, VLSI use
